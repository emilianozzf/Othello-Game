4.14 Exercises Questions 1-7
1. What possible values can a Boolean expression have?
A: A Boolean expression can have values either True or False.

2. Where does the term Boolean originate?
A: The term “Boolean” comes from the man who invented Boolean Logic in the 19th century – George Boole.

3. What is an integer equivalent to True in Python?
A: The integer 1 is equivalent to True in Python.

4. What is the integer equivalent to False in Python?
A: The integer 0 is equivalent to False in Python.

5. Is the value -16 interpreted as True or False?
A: The value -16 interpreted as True in Python.

x, y, z = 3, 5, 7
evaluate the following Boolean expressions:
(a) x == 3
(b) x < y
(c) x >= y
(d) x <= y

(h) x < 0 and x < 10
(i) x >= 0 and x < 2
(j) x < 0 or x < 10
(k) x > 0 or x < 10
(l) x < 0 or x > 10

A:
(a) True
(b) True
(c) False
(d) True
(e) False
(f) True
(g) True
(h) False
(i) False
(j) True
(k) True
(l) False
